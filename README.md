# 🦁 1주차 - HTML/CSS 기초 (아기 사자 자기소개 페이지 만들기)

자바스크립트(JavaScript)를 사용하지 않고, 오직 HTML과 CSS만을 활용하여 나를 소개하는 ‘정적 자기소개 카드’ 화면을 구현합니다. 단순히 예쁜 화면을 만드는 것을 넘어 **'정보 - 구조 - 스타일'의 흐름**으로 웹 화면이 어떻게 구성되고 표현되는지 경험적으로 이해하는 것을 목표로 합니다.

## 🎯 과제 목표

* **웹의 3요소 이해**: HTML(구조), CSS(스타일), JS(상호작용) 중 구조와 스타일의 역할 분리 이해
* **정보의 구조화**: 프로필 이미지, 이름, 역할 등의 데이터를 시맨틱(Semantic) HTML 태그를 사용해 의미에 맞게 구조화
* **스타일 설계 및 레이아웃 구성**: 클래스(Class) 단위로 스타일을 설계하고, Flexbox를 활용해 카드 형태의 레이아웃 배치

## 🛠 기술 스택

* **Language**: HTML5, CSS3
* **Environment**: Web Browser, Code Editor (VS Code 권장)

## 📋 주요 기능

1. **프로필 정보 구조화**: 자기소개에 필요한 요소(사진, 이름, 소속, 한줄 소개 등)를 HTML 태그로 마크업합니다.
2. **카드 레이아웃 구현**: CSS Flexbox를 활용하여 프로필 요소들을 깔끔한 카드(Card) 형태로 배치합니다.
3. **시각적 스타일링**: 색상, 폰트, 여백(Margin/Padding) 등 CSS 속성을 활용하여 디자인을 완성합니다.

## 🚀 실행 흐름 (Implementation Logic)

1. **구조화 (HTML)**: 화면에 들어갈 콘텐츠를 정의하고 적절한 HTML 태그로 감싸 뼈대를 만듭니다.
2. **클래스 부여**: CSS로 스타일을 쉽게 제어할 수 있도록 각 HTML 요소에 목적에 맞는 `class` 이름을 부여합니다.
3. **스타일링 (CSS)**: 작성한 `class`를 선택자로 사용하여 크기, 색상, 정렬(Flexbox) 등의 CSS 속성을 적용하여 화면을 완성합니다.

## ⚠️ 제약 사항 및 유의 사항

* 이번 과제에서는 JavaScript를 절대 사용하지 않습니다.
* `id` 선택자보다는 재사용성이 높은 `class` 선택자를 위주로 스타일을 작성해 보세요.
* 브라우저 크기를 변경해도 레이아웃이 크게 깨지지 않도록 여백과 정렬을 신경 써주세요.
